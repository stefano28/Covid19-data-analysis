# Covid19 data analysis


## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to nstall dependencies

```bash
pip install plotly
pip install numby
pip install dash_bootstrap_components
pip install dash
```

## Usage

```bash
python app.py
```
Go to http://127.0.0.1:8050/ with your browser
